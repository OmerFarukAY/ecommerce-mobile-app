## 📌 About the Project
This repository contains a fully functional **E-Commerce Mobile Application** built with Flutter. While it starts with a secure and beautifully designed authentication flow, the core of the application reveals a comprehensive shopping experience. 

The project is designed with a focus on modern UI/UX principles, seamless navigation, and robust state/data management for a complete end-to-end user journey.

## ✨ Key Features
* **🛍️ Shopping Experience:** Intuitive product browsing, detailed product views, and cart management.
* **🔐 Authentication Flow:** Secure Login and Registration screens with password visibility toggles and smooth transitions. Session persistence is smoothly handled using `SharedPreferences`.
* **👤 Dynamic Profile Management:** A dedicated profile section where users can update their personal details (name, email, phone) and upload profile photos (utilizing Base64 encoding) via a centralized `AuthService`.
* **🎨 Modern Aesthetics:** Gradient-heavy interfaces, curved headers, and custom gradient icons for a consistent, premium feel across the entire app.
* **📱 Responsive & Interactive:** Modal bottom sheets, dynamic button state colors, and responsive layouts that adapt to various mobile screen sizes.

## 🛠️ Technologies & Architecture
* **Framework:** [Flutter](https://flutter.dev/)
* **Language:** Dart
* **Local Storage:** `SharedPreferences` for managing user sessions securely.
* **Architecture:** Modular component-based design, separating UI layers from service logic (e.g., AuthService).

## 🚀 Getting Started

To run this e-commerce project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash

   cd flutter-ecommerce-app

2. **Install dependencies:**
   ```bash
   flutter pub get

3. **Run the app:**
   ```bash
   flutter run
